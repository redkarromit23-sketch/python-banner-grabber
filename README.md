# Python Banner Grabber

A Python-based Banner Grabber that connects to a target host and port, retrieves server banners, and displays service information.

## Features

- TCP socket connections
- Banner grabbing
- Service identification
- User-defined IP and Port input
- HTTP banner retrieval

## Technologies Used

- Python
- Socket Programming
- Networking

## Example Usage

```bash
python banner.py
```

Example:

```text
Enter IP: 127.0.0.1
Enter Port: 8000

HTTP/1.0 200 OK
Server: SimpleHTTP/0.6 Python/3.14.2
```
